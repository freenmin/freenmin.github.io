Заметки с прочитанного

Операторы.

Сами статичные импорты появились в Java SE5. Благодаря статическому импорту появляется возможность ссылаться на статические члены непосредственно по именам, не угочняя их именем класса. Давайте разберем, где использовать статичные импорты уместно, а где - нет. Например нам нужно вывести на экран большой-большой текст, что мы выберем? System.out.println, или же заранее созданный метод print в классе "PrintUtils" который выводит текст на экран не хуже System.out.println, но занимает меньше места? Конечно же print, т.к он занимает меньше место, но если текст маленький, что тогда выбрать? Мой ответ: System.out.println. Давай-те же посмотрим примеры. PrintUtils (Класс с методом print):

package ru.boss90.myfirstapp.utils;

public class PrintUtils {

     public static void print(String text) {
          System.out.println(text);
     }

}

InfoAboutFox (Демонстрация import static):

package ru.boss90.myfirstapp.main;

import static ru.boss90.myfirstapp.utils.PrintUtils;

public class InfoAboutFox {

    public static void main(String[] args) {

     System.out.println("Лиса - хищник"); //Маленький текст, уместнее будет использовать System.out.println.
     print("Лиса – млекопитающий хищник небольших размеров. В биологической классификации      многочисленный род лисиц      относят к семейству псовых.."); //Большой текст, уместнее будет      использовать метод с класса PrintUtils.

     }

}

Увеличение и уменьшение.

Существуют 4 вида увелечение/уменьшенее операторов, а именно Префиксный инкремент, Постфиксный инкеремент, Префиксный декремент, и Постфиксный декремент. Что за эти ваши "Префиксный и Постфиксный"? Префиксный - перед переменной или выражением. Постфиксный - после переменной или выражением. Что такое Декремент и Инкеремент? Декремент - минусы, т.е --. Инкремент - плюсы, т.е ++. [Вопрос] Теперь мы узнали виды увелечении и уменьшее операторов, но в чем отличие между Постфиксный и Префиксный? Поставлю префиксальным видом, ну и что? А захочу, поставлю и постфиксным видом, в чем разница? Между ними разница достаточно влиятельная. Префиксный Инкремент и Декремент (т.е ++i, --i) сначала выполняется операция, а затем выдается результат. А вот для постфиксной записи (i++, i--) сначала выдается значение и лишь потом выполяется операция. Например:

public class Operators {

    public static void main(String[] args) {

        i=1;
        System.out.println("i="+i); //Вывод: 1.
        System.out.println("++i="++i); //Вывод 2.
        System.out.println("i++="i++); //Вывод 2, но сама переменная i уже значит 3, а не как раньше 2.
        System.out.println("i="+i); //Вывод 3.


    }

}

Термины

ЯП - Язык программирования.

Junior - Начальный уровень программиста, с опытом работы до 2 лет, имеющий МИНИМАЛЬНЫЙ опыт.

Middle - Средний уровень программиста, с опытом работы от 2 лет, имеет навыки в какой-либо сфере.

Senior - Высокий уровень программиста, с опытом работы от 5 лет, имеет навыки в какой-либо сфере.

API - совокупность функции, которые помогают обращаться к сервису, который предоставил API. Получить местоположение посылки, отправить посылку, получить сообщение, отправить сообщение, и тп.

Front-End - Работа с WEB частью в плане дизайна, красоты сайта.

Back-End - Работа с WEB частью в плане самого кода, системы, создание API, и тп.
